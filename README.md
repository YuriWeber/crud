# CRUD--REACTJS-MYSQL

Sistema com login, registro e gerenciamento de contas de usuários.

## Instalação
  ### Front-end
    $ cd client
    $ npm install
    $ npm start
  ### Back-end
    $ cd server
    $ npm install
    $ npm run dev
    
## Funcionalidades
  Conectar e cadastrar como usuário
  Editar conta
  Visualizar e gerenciar contas dos usuários como administrador
  Deletar contas
  

## Tecnologias:
  ### FRONT-END
    HTML,
    CSS,
    JavaScript,
    React
      Material-UI
      Axios
  ### BACK-END
    Node.js,
      express,
